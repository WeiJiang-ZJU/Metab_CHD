# Resolving Early Targets and Metabolomic Mechanisms of Congenital Heart Disease Through Tandem Mass Spectrometry Screening in Neonates

## Overview
This repository contains the R code and Python code associated with the research article titled "Resolving Early Targets and Metabolomic Mechanisms of Congenital Heart Disease Through Tandem Mass Spectrometry Screening in Neonates". The article is currently under submission, and relevant data will be available upon reasonable request after publication.

## Requirements
- R version 4.4.1 or later
- Python 3.9.12 or later

## Installation
To run the code, ensure you have the required R and Python version installed and load any necessary libraries as indicated in the scripts.

## Usage
1. Clone this repository to your local machine
2. Navigate to the repository directory
3. Open R and set your working directory to the repository path
4. Run the provided scripts in the desired order to reproduce the results of the study.
